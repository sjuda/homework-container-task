

## deque
Реализуйте аналог контейнера std::deque. Продемонстрируйте работу вашего класса (необходимо вызвать каждый метод). Проверьте ваш код на утечки памяти. Реализуйте юнит-тесты к классу.
```cpp

template <class T>
class deque
{
public:
    кострукторы
    декструктор
    оператор присваивания
    
    begin
    end
    
    empty
    size
    
    front
    back
    
    push_front
    pop_front
    push_back
    pop_back
    insert
    erase
    clear
    swap
};
```
