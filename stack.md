## stack
Реализуйте аналог контейнера std::stack. Продемонстрируйте работу вашего класса (необходимо вызвать каждый метод). Проверьте ваш код на утечки памяти. Реализуйте юнит-тесты к классу.

```cpp
template <class T>
class stack
{
public:
    кострукторы
    декструктор
    оператор присваивания
    
    empty
    size
    
    push
    pop
    top
    
    swap
};
```
