## queue

Реализуйте аналог контейнера std::queue. Продемонстрируйте работу вашего класса (необходимо вызвать каждый метод). Проверьте ваш код на утечки памяти. Реализуйте юнит-тесты к классу.
```cpp
template <class T>
class queue
{
public:
    кострукторы
    декструктор
    оператор присваивания
    
    empty
    size
    
    push
    pop
    front
    
    swap
};
```
